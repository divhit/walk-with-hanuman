# Walk with Hanuman

The Ramayana, told to you. A voice-first storybook where Hanuman (an ElevenLabs
conversational agent) tells the whole epic out loud in five chapters — and at
every hard choice stops to ask what _you_ would do. The screen is a painted
storybook scene that changes as the story moves.

## The five chapters

1. **The Four Princes** — Ayodhya, Vishwamitra's demand, Tataka, Shiva's bow, the wedding
2. **The Promise** — Kaikeyi's boons, the exile, Bharata and the sandals
3. **The Golden Deer** — Panchavati, the rekha, the abduction, Jatayu
4. **The Great Leap** — meeting Rama, Vali, "remember who you are", the leap, finding Sita
5. **The Bridge and the Lamps** — Vibhishana, the squirrel and the bridge, the sanjeevani mountain, Ravana's fall, the first Diwali

Each chapter carries three genuine dilemmas the child talks through with
Hanuman before he reveals what the characters chose. One agent tells all five —
the app passes `chapter_num` / `chapter_title` as dynamic variables.

## Stack

- Vite + React, plain CSS (Rozha One + Baloo 2 via Fontsource)
- `@elevenlabs/client` — WebRTC conversation with the Hanuman agent
- Scene art: painted storybook images in `public/scenes/` generated with
  gemini-3-pro-image (`scripts/generate-scenes.py` + `generate-chapter-scenes.py`,
  hero image as style anchor); SVG tableaux in `src/scenes.jsx` back the 3D world
- The agent drives the picture book: a `show_scene` client tool switches scenes
- Chapter data + progress (localStorage): `src/chapters.js`
